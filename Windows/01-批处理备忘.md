## 文件路径相关处理
下面的批处理脚本可以获取文件所在路径、文件名、文件后缀名等信息。
>注意要用 GBK 或者 GB18030 编码。
```batch
@echo off

set /p filepath="Please input file path: "
echo %filepath%
for %%f in (%filepath%) do (
    set filename=%%~nf
    set extname=%%~xf
    set dirname=%%~dpf
    set fullname=%%~nxf
    set noextpathname=%%~dpnf
)

echo 文件名:%filename%
echo 后缀名:%extname%
echo 所在目录:%dirname%
echo 完整文件名:%fullname%
echo 无后缀名路径:%noextpathname%
pause
```

- 输出示例1：
```
Please input file path: D:\Download\Country.mmdb
D:\Download\Country.mmdb
文件名:Country
后缀名:.mmdb
所在目录:D:\Download\
完整文件名:Country.mmdb
无后缀名路径:D:\Download\Country
请按任意键继续. . .
```
